# 🎬 Movie Recommendation System

This project recommends similar movies using Machine Learning.

## 🚀 Features
- Content-based recommendation
- Uses cosine similarity
- Built with Python

## 🛠 Tech Stack
- Python
- Pandas
- Scikit-learn

## 📌 How it works
- Converts movie data into vectors
- Finds similarity between movies
- Recommends top 5 similar movies
